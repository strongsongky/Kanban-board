# Kanban-board

#### [ 개요 ]
- CRUD가 가능한 To-do list에 드래그 앤 드랍으로 이동 가능한 칸반 보드 형식을 적용시킨 웹페이지
- 추가 버튼을 클릭하여 할 일 카드를 추가
- 각 카드는 다른 칼럼으로 이동하거나 수정 및 삭제 가능
